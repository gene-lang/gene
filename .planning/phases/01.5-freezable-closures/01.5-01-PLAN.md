<objective>
Implement the core Phase 1.5 runtime rule: `(freeze v)` must accept `VkFunction`
closures when their captured `Scope` graph is itself freezable, and must reject
non-freezable captured state before any `deep_frozen/shared` bits are set.

Purpose: Close the last substrate gap before Phase 2 actor send can treat
callable payloads like other frozen values.
Output: A closure-aware freeze walker plus focused regression coverage.
</objective>
